## Об аудиогиде

Мы посетили Givskud Zoo — очень впечатляющий зоопарк, который я однозначно рекомендую. Особенно интересен автомобильный маршрут: вы едете на своей машине, а животные находятся прямо вокруг вас. Больше всего нас впечатлила зона со львами.

У зоопарка есть бесплатное приложение-аудиогид, но русского языка в нём нет. Наиболее подходящим вариантом для нас был английский, однако мой сын понимает его недостаточно хорошо.

Поэтому я подготовил неофициальную русскую версию Android-приложения с переводом, выполненным с помощью ИИ.

В сборке:

- сохранена оригинальная логика приложения;
- работают GPS-зоны;
- аудиозаписи автоматически переключаются во время движения по парку;
- переведены 33 аудиозаписи;
- добавлен русский язык в переключатель;
- частично переведён интерфейс приложения _(основная цель проекта — русскоязычное аудио с сохранением GPS-логики)_.

Интонации синтезированного голоса местами неидеальны, но аудиогид позволил нам узнать много интересных фактов о животных и сделал поездку заметно увлекательнее.

## 📱 Установка

1. Удалите официальное приложение Givskud Zoo, если оно установлено.
2. [Скачайте APK](https://github.com/LanKing/Givskud-Zoo-Denmark-Russian-Audioguide/releases/latest/download/Givskud_Zoo_RU.apk).
3. Разрешите установку приложений из неизвестных источников.
4. Установите APK.
5. В приложении выберите язык **«Русский»**.

Официальное приложение необходимо удалить из-за различия цифровых подписей. Установить эту версию поверх приложения из Google Play не получится.
Сборка основана на Givskud Zoo `1.3.4` и предназначена для Android-устройств с архитектурой ARM64.
